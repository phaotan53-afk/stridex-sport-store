# STRIDEX SPORT - Website ban do the thao

Cong nghe:
- Frontend: Angular + Tailwind CSS + Angular Animations
- Backend: ASP.NET Core Web API (.NET 8)
- Database demo: InMemory, co the doi sang SQL Server

## Cach chay Frontend
```bash
cd frontend-angular
npm install
npm start
```

## Cach chay Backend
```bash
cd backend-dotnet
 dotnet restore
 dotnet run
```

API mac dinh: http://localhost:5000
Frontend mac dinh: http://localhost:4200
